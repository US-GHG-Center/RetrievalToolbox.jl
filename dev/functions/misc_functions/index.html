RetrievalToolbox.findany — Method
findany(x::AbstractVector, a) -> Bool

Returns true if a is found within x, does not allocate.

Examples

julia> findany(["a", "b", "c", "d"], "b")
true
julia> findany([1,2,3,4,5], 10)
false
